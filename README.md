# 计算机辅助翻译系统
软件工程理论与实践课程设计

## 项目概述
基于Vue2+Django的辅助翻译系统，提供文件上传、多项目管理、翻译记忆库和术语库等功能。

## 技术栈
| 组件       | 技术选型                          |
|------------|-----------------------------------|
| 前端框架   | Vue 2.6 + Vuex 3.6 + Vue Router 3.5 |
| UI组件库   | Element UI 2.15 |
| HTTP客户端 | Axios 1.8 |
| 后端框架   | Django 5.1.3 |
| 数据库     |  MySQL 8.0 |
